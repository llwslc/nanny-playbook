# 民间陋习清单

> 年纪大一点的阿姨，**默认动作里全是这些**。不是她坏，是她那辈人就这么带孩子的。
> 所以不能靠"她答应了"来解决——要靠**问出来** + **写进红线** + **摄像头**。

---

## 怎么问（这一节比清单本身重要）

**❌ 别问是非题**

> "你会不会把尿？"

她知道标准答案是"不会"。**你什么都问不出来。**

**✅ 用预设式** —— 把陋习当成默认前提，让她在你的前提里回答

| 问 | 在测什么 |
|---|---|
| "你之前带的孩子，**一般几个月开始把尿**？" | 把屎把尿 |
| "孩子**多大开始喂水**的？" | 6 个月内喂水 |
| "孩子发烧了，你**怎么**帮他降温？" | 捂汗、酒精擦浴 |
| "孩子便秘了，你一般怎么办？" | 开塞露、肥皂条 |
| "孩子**多大开始加盐**的？菜里放不放酱油？" | 1 岁内加盐 |
| "**头型**怎么睡好看？" | 硬枕头、定型枕 |
| "孩子哭得厉害停不下来，你怎么哄？" | 摇晃、抛高高 |
| "孩子**新生儿那会儿**，乳头要不要处理一下？" | 挤新生儿乳头 |
| "你一般**多久给孩子捏一次脊**？" | 捏脊 |
| "孩子**有点咳嗽 / 积食**，你一般怎么给他调理？" | 拿小儿推拿**当治疗** —— 延误就医 |
| "**抚触**你具体做哪几样？从头到脚说一遍。" | "抚触"是个筐：捏脊、挤乳头、绑腿、割螳螂嘴，全能装进去 |

**✅ 或者开放式** —— 不给方向，看她第一个想到什么

**⚠️ 绝对不要给暗示**

> "我们家是不让把尿的哦，你会把尿吗？"

——这题白问了。

## 清单

### 喂养

| 陋习 | 危害 |
|---|---|
| 6 个月内喂水 / 葡萄糖水 / 茶水 / 米汤 | 占肚子、影响奶量；过量会**水中毒** |
| **1 岁内加盐、加糖** | 肾脏负担、口味重 |
| **1 岁内喂蜂蜜** | **肉毒杆菌中毒** |
| **嚼碎了喂 / 嘴对嘴喂 / 用嘴试奶温 / 共用餐具** | **幽门螺杆菌、疱疹病毒**（对新生儿可能致命） |
| 用奶瓶喂米糊 | 呛咳、过量 |
| 追着喂 / 逼吃 / 强迫喂 | 厌食、进食创伤 |
| 给孩子喝大人的汤、粥、饮料 | 含盐、含糖——加重肾脏负担、养出重口味；汤汤水水还占肚子，挤掉奶和正经辅食 |
| 冲奶**先粉后水** | 浓度不准 |
| 奶粉**多舀一勺"长得快"** | 肾脏负担、便秘 |
| 剧烈摇晃奶瓶 | 大量气泡 → 胀气、肠绞痛 |

### 护理

| 陋习 | 危害 |
|---|---|
| **把屎把尿** | 髋关节发育不良、脱肛、盆底肌松弛 |
| **挤新生儿乳头** | 乳腺炎、感染 |
| **挑马牙 / 挑鹅口疮** | 感染 |
| **割螳螂嘴**（口腔里的吸吮脂肪垫） | **出血、感染** —— 那是正常的，是帮他吸奶用的 |
| **捏脊** | 没有证据。危害见「生病」那一组——**它会替代该做的事** |
| **用力按 / 揉囟门** | 那里没有颅骨，只隔着一层膜。正常洗头、轻轻碰都没事，**但不能使劲按揉** |
| 剃胎毛 / 剃眉毛 / 刮头 | 毛囊感染 |
| **绑腿 / 蜡烛包**（"防罗圈腿"） | **髋关节发育不良** |
| 硬枕头 / 书本 / 绿豆枕 睡头型 | 扁头、斜颈 |
| **定型枕、任何枕头**（1 岁内） | **窒息** |
| 戴银锁、银手镯 | 勒伤、误吞、划伤 |
| 穿开裆裤 | 细菌、外伤 |
| 长指甲不剪 / **戴手套防抓脸** | **线头缠指坏死** |
| 掏耳朵、挖鼻孔 | 损伤 |
| 湿疹不涂保湿、只用偏方 | 湿疹护理的核心就是保湿 |

### 生病（危害最大的一组）

| 陋习 | 危害 |
|---|---|
| **捂汗退烧** | **捂热综合征——真的能捂死人** |
| **酒精擦浴** | **婴儿经皮吸收，酒精中毒** |
| 冰水澡 / 冰敷 | 寒战反而升温 |
| **自行喂退烧药、抗生素** | 剂量、禁忌（阿司匹林 / 安乃近 / 尼美舒利） |
| 喂"七星茶""猴枣散"**"保婴丹"** | **保婴丹含朱砂，即硫化汞** |
| 开塞露 / 肥皂条通便 | 依赖、黏膜损伤 |
| **小儿推拿 / 捏脊"退烧""治积食""治咳嗽"** | 没有可靠证据能治病。**真正的危险是它替代了就医** —— 推着推着，该去医院的时间过去了 |
| **孩子病了或磕了，瞒着不说** | **最危险的一条**——延误的是抢救时间 |

### 安全

| 陋习 | 危害 |
|---|---|
| **摇晃哄睡 / 抛高高 / 举高高转圈** | **婴儿摇晃综合征**——脑出血、致残致死 |
| **学步车** | 髋关节；翻倒、坠楼（多国已禁售） |
| 拽胳膊把孩子提起来 | **牵拉肘**（桡骨头半脱位） |
| 抱着孩子坐副驾 / 不用安全座椅 | 车祸时孩子就是安全气囊的缓冲垫 |
| **把孩子单独留在床上、沙发、浴盆、车里** | 坠落、溺水、中暑——**哪怕一秒** |
| 给孩子玩塑料袋、**纽扣电池、磁力珠** | 窒息、烧穿食道、肠穿孔 |
| 3 岁前给整颗坚果 / 果冻 / 爆米花 / 整颗葡萄 | **窒息**（6–12 个月是异物窒息最高危期） |
| 叼着奶瓶睡 | 奶瓶龋、呛奶 |
