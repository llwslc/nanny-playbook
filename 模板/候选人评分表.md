# 候选人评分表

> **面试完立刻填，别靠回忆。** 见了三个人之后，你一定会记混。
> ⚠️ 填好的版本含个人信息，**不要提交到公开仓库** —— 放 `本地/` 或仓库外面。

| 维度 | 分值 | 怎么打 | A | B | C |
|---|---|---|---|---|---|

## 第三步：不打分，但要写下来

这几项**不要分数化**。它们经常比分数准。

| 观察项 | A | B | C |
|---|---|---|---|
| **孩子的反应**——在她怀里身体是软的还是僵的？她靠近时孩子什么表情？ | | | |
| **她看谁**——全程盯着你说话，还是会自然地看向孩子？ | | | |
| **老人的感觉**（老人要跟她朝夕相处，老人的直觉值钱） | | | |
| **伴侣的感觉** | | | |
| 你自己说不上来但觉得不对的地方 | | | |

> **如果分数很高但家里人都觉得不对劲——听家里人的。**
> 分数只能量化"她会不会"，量化不了"她是个什么人"。
