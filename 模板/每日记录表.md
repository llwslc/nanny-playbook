# 每日记录表：用法

> 空白打印版 → [每日记录表-空白.md](每日记录表-空白.md)

---

## 为什么必须有这张表

不是形式主义，是四件事：

**1. 发现问题**
奶量连续下降、尿量不足、大便颜色异常——这些是**婴儿生病最早的信号**，而且**只有记下来才看得出趋势**。凭记忆是看不出"这两天奶量少了 20%"的。

**2. 留痕**
出了事，表是唯一的客观记录。

**3. 保护她**
孩子生病了，表能证明她做了该做的事：喂了多少、什么时候发现的异常、什么时候报告的。

> **这一点要当面讲给她听。** 她才不会觉得你在防贼。

**4. 交接**
你、老人、她之间靠表说话，不靠"我以为你喂过了"。

## 表格（带示例）

### A. 喂奶（每次一行，当场填）

| 时间 | 奶别 | 计划量 | 实喝 | 拍嗝 | 吐奶 | 备注 |
|---|---|---|---|---|---|---|
| 06:20 | 配方奶 | 120 ml | 100 ml | ✓ 打出 | 无 | |
| 09:10 | 母乳亲喂 | 左 10 分 右 8 分 | — | ✗ 没打出 | 少量溢奶 | 有点闹 |

- **奶别**：母乳亲喂 / 母乳瓶喂 / 配方奶
- **计划量**：配方奶填 ml；母乳亲喂填左右各几分钟
- **拍嗝**：`✓ 打出` / `✗ 没打出`
- **吐奶**：`无` / `少量溢奶` / `大量` / **`喷射状` 🚨 立即打电话**

> **日终小计：喂奶 ___ 次，总奶量 ___ ml**

### B. 大小便

| 时间 | 小便 | 大便 | 性状 | 颜色 |
|---|---|---|---|---|
| 08:15 | ✓ | — | | |
| 11:40 | ✓ | ✓ | 糊状 | 黄 |

- **性状**：糊状 / 成形 / 稀水样 / 干硬球状
- **颜色**：黄 / 绿 / 褐 / **🚨 白陶土色** / **🚨 带血丝** / **🚨 黑色**

> **日终小计：尿不湿 ___ 片，大便 ___ 次**

### C. 睡眠

| 入睡 | 醒来 | 时长 | 哄睡方式 |
|---|---|---|---|
| 09:00 | 10:30 | 1h30m | 拍睡 |

> **日终小计：白天睡 ___ 小时，夜间睡 ___ 小时，夜醒 ___ 次**

### D. 体温 / 用药 / 异常

| 项 | 内容 |
|---|---|
| **体温**（仅异常时测） | 时间 ___　___ ℃　部位 ___ |
| **用药** | 药名 ___　剂量 ___　时间 ___　**家长事先同意签字：______** |
| **异常事件** | 磕碰 / 坠落 / 呛咳 / 皮疹 / 哭闹不止 / 其他：___<br>**已电话报告：□ 是（时间 ___）　□ 否** |

> **用药栏没有家长签字 = 违反[红线](红线清单.md)第 2 条，立刻解约。**

### E. 每日确认

> 育儿嫂签字：______　家长确认：______　日期：______

## 填写规则（写进合同）

1. **当场填，不许事后补。** 喂完一顿填一行
2. **每天下班前（或睡前）拍照发家庭群**
3. **计划奶量由家长 / 儿保医生定，她只如实记录实喝量。** 她不能自作主张加量减量
4. **家长每天签字确认** —— 这既是核对，也是给她的凭据
5. **🚨 出现下面任何一项，立刻打电话，不等下班：**

| 信号 | |
|---|---|
| 吐奶呈**喷射状** | |
| 大便**白陶土色 / 带血 / 黑色** | 顺便拍照 |
| **一天尿不湿少于 6 片** | 可能脱水 |
| **奶量比平时少三分之一以上** | |
| 体温异常 | |
| 任何磕碰、坠落 | |

## 怎么从表里看出问题

**这张表的价值在趋势，不在单行。**

| 表上的信号 | 可能是什么 |
|---|---|
| 奶量连续 2–3 天下降 | 生病前兆（感染、鹅口疮、中耳炎） |
| **尿不湿 < 6 片 / 天** | **脱水** |
| 大便由黄转**白陶土色** | **胆道闭锁——急症，有手术时间窗，越早越好** |
| 大便带血 / 黑色 | 消化道出血、过敏 |
| **喷射状吐奶** | 幽门狭窄 / 颅内压高 |
| 拍嗝一栏**总是"✗ 没打出"** | 她的拍嗝手法有问题，回炉 |
| 夜醒次数**远多于**她口头说的 | 她在虚报夜班工作量，或哄睡方式有问题 |
| **「睡得久」+「奶量降」+「尿不湿变少」三条同时出现** | **孩子可能被喂了镇静药。** 见下 |

### ⚠️ 「睡得久」单独一条，什么都不说明

婴儿的睡眠**本来就会随月龄变长**——这是发育，不是异常。
（基线 → [参考/标准答案.md](../参考/标准答案.md) §睡眠「几个月能睡整觉」）

**但发育性的睡眠延长，奶量是稳的、甚至是升的。**

**镇静药不只让孩子睡——它连带压制吃、尿、拉。** 所以：

| | 发育性（正常） | **药物性（危险）** |
|---|---|---|
| 睡眠变长的速度 | 以**周**为单位，渐进 | 以**天**为单位，**突然** |
| **奶量** | **稳定或上升** | **下降** |
| **尿不湿** | 稳定（≥6 片） | **变少** |
| 醒来的状态 | 精神好、马上找吃的 | **懵、软、叫不太醒、吸吮无力** |

> **这就是这张表存在的意义**——没有它，你根本看不出这三条线的走向，只能凭"感觉孩子最近睡得多"瞎猜。

完整的判据（生理体征、摄像头怎么看、事前五道防线）→ **[参考/危险信号.md](../参考/危险信号.md) §怎么判断孩子被喂了药**

## 怎么识别"编的表"

**真实的一天是乱的。太干净的表本身就是红旗。**

| 症状 | 说明 |
|---|---|
| **时间整齐得不像话** —— 每次喂奶都是整点，间隔精确 3 小时 | 真实的婴儿不这样 |
| **实喝量永远等于计划量** —— 120 计划、120 喝完，天天如此 | **几乎不可能** |
| **"异常"栏永远空白**，一个月都没填过一次 | 真实的一天总会有点什么 |
| **笔迹和墨色完全一致** | 一天七顿奶如果是当场填的，笔迹会有深浅、有涂改。**一模一样 = 下班前一口气补的** |
| **跟摄像头对不上** | 最硬的一招：随机抽一天，拿表去对回放，看喂奶时间点对不对得上 |

> **发现一次编表，按隐瞒处理** —— 和[红线](红线清单.md)第 12 条是同一件事。

## 她写不来怎么办

文化程度低但踏实的阿姨很多，**别一刀切**。

改方案：**打勾式表格 + 语音报数** —— 她每次喂完在群里发一条语音（"六点二十，一百二十毫升，喝了一百，嗝打出来了"），你或老人代填。

**关键是留痕，不是她亲手写。**

但有一条不能商量：

> **"我记性好，不用记" —— 一票否决。**

---

## 最后：你得看

**你不看，她三天就不填了。**

每天她发上来，**回一句**——哪怕只是"收到"。这一句是整个机制能不能活下去的关键。
