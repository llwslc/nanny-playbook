# nanny-playbook

找育儿嫂的一整套剧本：怎么筛、怎么验、怎么问、怎么试、签什么、上岗后怎么管。

**六道关，逐关淘汰。** 一次面试问不出人品。

---

## ⚠️ 这是公开仓库——不要往里放任何个人信息

你会收到**征信报告**（身份证号 + 住址 + 全部信贷账户）、**体检报告**（医疗数据）、**无犯罪记录证明**。

**候选人材料一律存在这个仓库之外。**

> **git 历史是永久的。** 提交一次，`git rm` 掉也还在历史里、还在每一个 clone 里、还在 GitHub 的缓存里。公开仓库更是直接对全世界可见。

`.gitignore` 里有一层兜底（`*.pdf` `*.jpg` `*身份证*` …），但那是防手滑的最后一道，**不是让你把材料放进来的理由**。

同理，仓库里的表**都是空白模板**——[岗位定义.md](岗位定义.md)、[模板/候选人评分表.md](模板/候选人评分表.md)、[模板/每日记录表.md](模板/每日记录表.md)。
你填好的那些（孩子月龄、预算、常去的医院、候选人的名字和分数）**一律不要提交**——复制到 `本地/`（已 gitignore）或仓库外面再填。

## 面试之前：先读 [参考/标准答案.md](参考/标准答案.md)

**题库只有「问什么」，没有「什么是对的」。** 你不知道正确答案，就判断不了她答得对不对——她说"先粉后水"，你听不出问题。

而且**你是家长**：她休假、她辞职、她做错了，最后兜底的都是你。**这些东西你迟早要会。**

## 三个最容易被跳过、也最值钱的动作

1. **自己打电话给前雇主**，问一句：**"如果您再生一个，还会请她吗？"** —— 犹豫本身就是答案。
2. **让她把急救做出来给你看**，别接受口头描述。
3. **带薪试工 3–7 天。** 面试可以准备，七天装不出来。

> **无犯罪记录证明也要开**（户籍地派出所），但要知道它管什么：
> 它只筛得掉**被判过刑的人**。虐待、瞒事、脾气暴、不负责——**一样都不会写在上面。**
> **它是地板，不是天花板。** 上面这三件，才是真正能照出人的。

## 两条贯穿始终的原则

**能看的都要看，能做的都要做。** 身份证、无犯罪、体检、征信、背调——**这五样都要**。这不是不信任，是给孩子上保险。

> 但要对：这些材料**只能由她本人提供**，你不能替她去查，也不能强迫她开。
> **她可以拒绝——而你可以因此不聘。** 这套东西就是这么运转的，不需要更硬。
> **法律不许作为聘用条件的项目**（比如乙肝检测），**别要，要了也站不住**。
> → [流程/2-材料核验.md](流程/2-材料核验.md)

**态度要平和。** 好阿姨也在挑雇主，上来像审犯人一样过一遍，好的直接就走了。红线讲清楚，但话要说明白：

> **这些规矩是为了孩子，也是为了保护你自己——出了事，白纸黑字、记录在案，对谁都是保护。**

## 免责声明

- **[参考/急救.md](参考/急救.md) 不能替代正规急救培训。** 它是用来**在面试中判断对方会不会**的对照表，不是让你照着学。真要用，去红十字会或医院上一门婴幼儿急救课——你和她都该上。
- 所有医学相关内容仅供参考，**具体判断以儿科医生为准**。孩子的情况以你的儿保医生说的为准。
- 用工形式、个人信息处理、摄像头安装的合规要求，各地不同，**以当地法规和你的律师意见为准**。摄像头**不得**安装在住家阿姨的卧室和卫生间。
- 这份指南不构成医疗、法律或用工建议。
