# 第三关：背调（最有效的一关）

> **要至少两家前雇主的电话，你自己打，不通过中介转达。**
> 这一关的信息密度比面试高得多——前雇主已经和她朝夕相处过几个月，你只有两个小时。

---

## 话术（从软到硬，别一上来就查户口）

**1. 破冰，先给她说好话的空间**

> "她在您家带的孩子多大？带了多久？"
> "她最擅长哪块？"

对方会放松，也会先说好的一面。**让她说完。**

**2. 对口径**

> "为什么离开的？"

和她本人说的对一遍。对不上，记下来，正式面试时追问。

**3. 用软话问硬事**

> "有没有什么地方是您当时觉得**可以更好**的？"

比"她有什么缺点"更容易问出真话——「可以更好」这个说法，给了对方一个体面的台阶。

**4. 杀手锏**

> **"如果您再生一个，还会请她吗？"**

**这一句最杀。犹豫本身就是答案。**

真心推荐的人会立刻说"会啊"。任何形式的停顿、"这个嘛……"、"看情况"，都要往下追。

**5. 考她会不会隐瞒**

> "她带的这段时间，有没有发生过什么意外？磕了碰了、生过病？她当时是怎么处理的？"

听的是：**孩子出事时，她第一时间是告诉家长，还是先想说辞。**

## 危险信号

| 信号 | 判读 |
|---|---|
| 电话打不通 / 是空号 | 电话是假的 |
| 对方支支吾吾，"还行吧""挺好的"就没了 | 不想说坏话，但也不愿意推荐 |
| "她人挺好的，就是……" 后面不说了 | **就是**后面那半句才是重点，追问 |
| 中介坚持"我们帮您问，您别自己打" | 中介在挡 |
| 每份工作都做不满 3 个月 | 见 [参考/危险信号.md](../参考/危险信号.md) |
| 只能提供中介出具的"好评" | 一文不值 |
