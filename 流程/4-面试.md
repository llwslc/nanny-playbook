> **打印这一页，照着问。**
> 老人如果在家常住，**必须参加面试**——真正跟她朝夕相处的是老人，不是你。

## 0. 提问方法（比题目本身更重要）

**别问是非题。** "你会不会把尿？"——她知道标准答案是"不会"。你什么都问不出来。

### ① 预设式——把陋习当成默认前提，让她在你的前提里回答

> "你之前带的孩子，**一般几个月开始把尿**？"
> "孩子**多大开始喂水**的？"

### ② 开放式——不给方向，看她第一个想到什么

> "孩子发烧了，你**怎么**处理？"（等她自己说出捂汗、酒精、退热贴）
> "孩子便秘了怎么办？"（等她说开塞露、肥皂条）

### ③ 情景追问——给具体场景，然后追到底

> "你一个人在家，孩子从床上掉下来了。**接下来五分钟你做什么？**"
> → "然后呢？" → "如果他吐了呢？" → "**如果我电话打不通呢？**"

### ④ 复述交叉——同一件事隔半小时再问一遍

看口径对不对得上。**撒谎的人第二遍会飘。**

### ⚠️ 绝对不要给暗示

> "我们家是不让把尿的哦，你会把尿吗？"

——这题白问了。

## 1. 履历真实性

- **按时间顺序口述**每一份工作：孩子多大、带了多久、住在哪个区、为什么走
- **空窗期必须交代清楚**
- 和 [3-背调.md](3-背调.md) 里前雇主说的对一遍
- 和征信报告里的"居住信息""职业信息"对一遍

**关于证书**：育婴师证 / 月嫂证参考价值很低（国家职业资格目录已取消育婴员，现在是社会机构发证）。

**但可以反过来用**：问"证在哪考的？培训了多久？学了什么？"——用来测她的真实水平。

## 2. 育儿硬技能

按月龄加权 → [参考/月龄重点.md](../参考/月龄重点.md)

**喂**

- **冲奶**：水温多少？**先放水还是先放粉**？多少水配几勺？
- 能不能多舀一勺"长得快"？（**不能**，肾脏负担）
- 奶粉开封多久内用完？（1 个月）
- **试奶温怎么试**？（滴手腕内侧。**用嘴尝的直接扣分**）
- 奶瓶怎么消毒？多久一次？
- **母乳储存**：冷藏多久？冷冻多久？怎么解冻？（**能用微波炉吗？——不能**）
- **拍嗝**：拍多久？拍不出来怎么办？（现场演示 → [5-实操测试.md](5-实操测试.md)）
- **吐奶 vs 呛奶**怎么区分？各自怎么处理？

**抱、换、洗**

- 新生儿怎么抱？几个月能竖抱？
- 换尿布：女宝宝怎么擦？（**从前往后**）红屁股怎么处理？（保持干燥 + 护臀膏，**不是撒爽身粉**）
- 洗澡：水温？先洗哪里？脐带没脱落怎么洗？耳朵进水怎么办？

**睡**

- 怎么哄睡？会不会抱睡、奶睡、摇睡？
- **孩子该怎么躺？**（仰卧）
- **床上能放什么？**（**什么都不能放**——枕头、被子、床围、毛绒玩具、定型枕，一样都不行）

**哭**

- **"孩子哭了，你第一步做什么？"**
  - ✅ 排查：饿了 / 尿了 / 困了 / 热了 / 不舒服
  - ❌ 直接抱起来摇
  - ❌ **"哭一会儿没事，锻炼肺活量"** → **一票否决**

**日常**

- 一天怎么安排？（会不会读绘本、做抚触、被动操，还是只管喂饱了事）
- **会不会给孩子看手机 / 电视？**

**辅食（到月龄再问）**

- 第一口吃什么？什么时候加？一次加几种？（**一种，观察 3 天**）
- **加不加盐？**（1 岁内**不加盐、不加糖、不加蜂蜜**）
- 过敏了怎么看、怎么办？
- 3 岁前不能给什么？（整颗坚果、果冻、爆米花、整颗葡萄、大块肉）

## 3. 急救 ⭐

> **这是唯一"答错了孩子会没命"的模块。**
> 标准答案 → [参考/急救.md](../参考/急救.md)。不会的，要么淘汰，要么上岗前你出钱送她去培训。

- **"孩子噎住了，你做给我看。"**（现场演示，别光说）
- 会不会婴儿心肺复苏？
- **孩子发烧，你怎么处理？**（开放式，等她自己说出捂汗 / 酒精 / 私自喂药）
- 多少度算发烧？什么情况必须立刻去医院？
- 摔下床 / 磕到头怎么办？
- 烫伤怎么办？
- 误食了东西怎么办？

**最重要的一题：**

> **"你一个人在家，孩子出事了。你做的第一件事是什么？"**

✅ **先急救 / 打 120，同时通知家长。**

这题真正考的是——她会不会因为怕担责任而**隐瞒、拖延、先想说辞**。

**再问：** 家附近哪家儿童医院？打 120 怎么报地址？

## 4. 民间陋习筛查

> **用预设式问，别用是非题。** 全清单 + 每一条的问法 → [参考/陋习清单.md](../参考/陋习清单.md)

现场至少排这几条（危害最大的）：

- **捂汗退烧 / 酒精擦浴**（能致死）
- **私自喂药**（尤其"保婴丹"含朱砂，即汞）
- **把屎把尿**
- **摇晃哄睡 / 抛高高**
- **嘴对嘴喂 / 用嘴试奶温 / 嚼碎了喂**
- **孩子病了或磕了，瞒着不说**

## 5. 卫生

- **进门第一件事做什么？**
  → **这条不用问。看她进门时怎么做。比十道题都准。**
- 接触孩子前洗手吗？做辅食前？上完厕所？
- 多久洗一次澡洗一次头？
- **会不会亲孩子？**（**红线**。成人携带的疱疹病毒对新生儿可能致命）
- **指甲——当场看。** 长指甲、涂指甲油、戴戒指手链的，是习惯问题，不是一句"我剪掉"能改的
- **抽烟吗？家里人抽烟吗？**
  - 本人抽烟 → **一票否决**
  - 家人抽烟 → 休息日回来必须洗澡换衣服（三手烟留在头发和衣服上）
- 化妆、喷香水吗？

## 6. 边界与冲突（决定她能待多久）

- **"我的做法和你的经验不一样，你怎么办？"**
  ✅ 听雇主的
  ❌ **"我带过多少个孩子了，你们年轻人不懂"** → 红旗

- **"和老人意见不合怎么办？"**
  住家阿姨最大的雷区。**所以老人必须在场，看她怎么当着老人的面回答这题。**

- **"上一份工作最不开心的是什么？"**
  **逢人就说前雇主坏话的，直接排除**——她下一个说的就是你。

- "被冤枉了（东西丢了、孩子磕了）你怎么办？"
- "孩子半夜闹觉，连着哭一个小时，你会烦吗？发过火吗？"
- "你觉得一个好雇主应该是什么样的？"（反向摸她的期待和雷点）
- "被辞退过吗？为什么？"

## 7. 作息与工作范围

对着 [岗位定义.md](../岗位定义.md) 逐条过：

- **夜里带不带睡？起夜几次？夜奶谁喂？**
- **除了带孩子，做不做饭？给几个人做？洗不洗大人的衣服？打扫到什么程度？**
  → **这一条说不清楚，三个月后必吵架**
- 一天作息：几点起、几点睡、午休多久（有些阿姨要求午休 2 小时）
- 月休几天？固定还是灵活？**春节回不回家？**
- **休息日回家后，回来必须洗澡换衣服**——接受吗？
- 加班怎么算？出差 / 旅行带不带她去？
- **手机**：带孩子的时候能不能刷手机、发语音？（红线）
- **能不能带老乡、亲戚、朋友进家门？**（红线：绝对不行）
- **摄像头**：公共区域 + 婴儿房，**不装她的卧室和卫生间**（装了违法）
  她可以有顾虑，但**强烈抗拒是个信号**
- **辞职提前多久通知？**（**至少 30 天**。不写这条，她哪天说走就走，你家直接瘫痪）

## 8. 每日记录 ⭐

> 这一条既是筛子，也是日后管理的抓手。表格 → [模板/每日记录表.md](../模板/每日记录表.md)

**原话这么问：**

> "我们家每天要填一张记录表——每次喂奶几点、多少毫升、实际喝了多少、拍嗝打出来没有、吐没吐；还有尿了几次、大便几次什么颜色、睡了多久。**每次喂完当场填，不能事后补。** 每天下班前拍照发到家庭群。
>
> 你之前填过吗？能填吗？"

**看反应：**

| 反应 | 判读 |
|---|---|
| 爽快答应，还能说出自己以前怎么记的 | 好 |
| "行，可以" | 正常 |
| 面露难色、含糊其辞 | **可能不识字或写不来数字**——用 [5-实操测试.md](5-实操测试.md) 测出来，**不是淘汰项** |
| **"我记性好，不用记"** | **一票否决。** 拒绝留痕的人，出了事你手上什么证据都没有 |
| "填这个太麻烦了 / 你们年轻人事真多" | **一票否决。** 这句话背后是"我不接受被监督" |

**再补一问：**

> "如果哪天记录表上出现异常——比如奶量突然少了一半，或者大便发白——你会怎么做？"

✅ **立刻电话告诉家长，不等下班。**

## 9. 家庭与财务背景

- 老家哪里？家里几口人？
- **她自己的孩子多大？现在谁在带？**（孩子太小 → 随时可能被叫回去）
- 配偶做什么？会不会经常来找她、在家门口等她？
- 父母身体怎么样？有没有大病？
- **未来一年有没有确定要请假的事？**（婚丧、盖房、老家农忙、孩子中高考）
- **有没有欠债？有没有网贷？有没有人催债？家里有人赌博吗？**
  （和征信报告对一遍 → [2-材料核验.md](2-材料核验.md)）
- **工资打你本人的卡吗？**（说要打老公 / 儿子卡的，**追问为什么**）
