## 体检：健康证只是地板，必须加项

家政"健康证"通常只查甲肝、痢疾、伤寒、活动性肺结核、化脓性皮肤病——**这是最低门槛，不够。**

要求：**三个月内、正规医院、你陪同或指定医院。**

在健康证基础上加：

| 项目 | 为什么 |
|---|---|
| 乙肝两对半 + 肝功能 | 基础 |
| 胸片（肺结核） | 密切接触，婴儿易感 |
| 梅毒、HIV | 基础 |
| **幽门螺杆菌（HP）** | 喂饭、试温、共餐传染。**最常被漏掉的一项** |
| **粪便培养**（沙门氏菌、志贺氏菌） | 只要她碰辅食就必须查 |
| 皮肤检查（疥疮、真菌、疱疹） | 皮肤接触传染 |
| 血常规 | 基础 |

再问一句：**最近有没有长期咳嗽？** 成人百日咳会传染给还没打完疫苗的婴儿。建议她每年打流感疫苗（你出钱）。

**上岗前约定好**：她一旦有呼吸道症状 / 腹泻 / 皮疹，立刻停止接触孩子。**谁来顶班、扣不扣钱，写进合同**——不然她会瞒着带病上岗。

## 征信报告怎么读

**要详版（明细版），不要简版。** 简版只有概要，没有明细和查询记录，看不出东西。

**获取渠道**（她本人操作，每年前两次免费）：

- 中国人民银行征信中心官网 `ipcrs.pbccrc.org.cn`
- 各家银行手机 App
- 云闪付 App
- 人民银行网点自助机

### 先说清楚为什么查

**不是查她穷不穷。** 白户（完全没有信贷记录）非常常见，**白户不是污点**。

查的是**现金流是否失控**——一个财务崩盘、被催债追着跑的人，在你家里接触现金、首饰、门禁、孩子，风险是实打实的。

这段话最好当面讲给她听，她才不会觉得受辱。

### ① 逾期及违约信息概要

- **呆账 / 核销 / 资产处置 / 保证人代偿** → **一票否决**
  这几个字的意思是"她真的还不上了"
- **当前逾期** → 高度警惕，问清楚
- 累计逾期看"**连三累六**"（连续 3 个月逾期，或累计 6 次逾期）——银行的坏账线，直接借用

### ② 信贷交易明细

- **看笔数，不看金额。** 网贷 / 小额贷公司**超过 3 笔**就危险
  多头借贷是现金流断裂的前兆，**比一笔大额房贷危险得多**
- 信用卡使用率长期 >90% → 拆东墙补西墙
- **担保记录**——她给别人担保过？这是或有负债。问清楚给谁担保的、对方还得上吗（农村家庭这种事很多）
- 还款记录是 **24 个月的状态码**：
  - `N` = 正常
  - 数字 `1`–`7` = 逾期档（`1` 是逾期 1–30 天，往上递增，`7` 是逾期 180 天以上）
  - **一排 N 里夹几个数字**，就是问题所在

### ③ 公共信息明细

- **强制执行、民事判决、欠税、行政处罚** → 有就一票否决，直接结束

### ④ 查询记录（最关键的一页，也最少人看）

- 看**近半年**的机构查询记录
- 查询原因写着"**贷款审批**""**信用卡审批**"的叫**硬查询**
- **半年内硬查询 ≥ 6 次 = 她在到处借钱**

> 哪怕一笔都没批下来、报告干干净净没有一条负债，**这一页也已经把真相写在脸上了**。

- "贷后管理""本人查询"不算硬查询，不用管

### ⑤ 顺手交叉验证

报告里有"**居住信息**""**职业信息**""**手机号**"——**跟她口述的简历对一遍**。对不上就追问。

### ⑥ 免费公开渠道，五分钟搞定

输姓名 + 身份证号就行：

- **中国执行信息公开网**——失信被执行人 / 被执行人
- **中国裁判文书网**——涉诉记录
